# Benchmark Results

This directory stores performance benchmarks and test results.

## Contents

- `benchmark_*.json` - Performance benchmark results
- `plots/` - Visualization charts (generated by test scripts)
- `metrics.txt` - Key performance metrics

## Running Benchmarks

```bash
# Run performance tests
docker compose up -d
./scripts/run_demo.sh

# View results
cat results/metrics.txt
```

## Key Metrics Tracked

- Intent recognition accuracy
- Response latency (p50, p95, p99)
- Knowledge base search time
- Conversation throughput
- Memory usage
