# Sample Datasets

This directory contains sample datasets for testing and demonstration.

## Contents

- `faq_samples.json` - Sample FAQ entries for knowledge base
- `chat_sessions.json` - Sample conversation sessions for testing
- `intent_training.json` - Training data for intent classifier

## Usage

These datasets are used by:
- The Lucene indexer for building the FAQ knowledge base
- The intent classifier for training
- Integration tests for validating conversation flows

## Format

All data files are in JSON format for easy parsing and integration.
